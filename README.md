# Rails Cashback API

## Description:
Developing a RoR fake cashback App for a awesome company as a test.

## Starting

### Requiriments

- Rails

### Running
```
> bundle install
> rails db:create && rails db:migrate
> yarn
> rails server
```
You can take a look in production server: https://fake-cashback-front.herokuapp.com/
```
> Admin: '/admin'
> Home: '/'
```
